Shows online users, glftpd status, stops/starts glftpd container and can be used
to view logs, edit config files and browse site. Also has a browser terminal
that displays tools like gl_spy, useredit and bot partyline (using websockets).

Cutting-edge tech used:

- PHP, some JQuery and Bootstrap4
- filemanager.github.io
- GoTTY web terminal
- pyspy (flask)
